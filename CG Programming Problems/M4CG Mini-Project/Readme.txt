To run main.cpp:

1. Open a new folder.

2. Open Dev-C++ and select <TDM-GCC 4.9.2 32-bit Release>.

3. Open a new project and select "Console Graphics Application". Write a name for the project and save.

4. Copy the code in main.cpp and paste to the current source file.

5. Compiile and run the program.

6. Select a transformation.

7. Input the coordinate for the vertices of a triangle as instructed. If "Rotate the Triangle" is chosen, input
   the coordinate for center of rotation after entering the coordinate for vertices.

8. Press "Enter" to see the result of transformation.
